📊 Student Performance Dashboard (Power BI Project)

📌 Project Overview
This project is a Student Performance Dashboard created using Microsoft Power BI. It helps in analyzing student data such as academic performance, attendance, and behavior in an interactive way.

🎯 Objectives
Analyze student performance across subjects and terms
Track attendance percentage
Monitor student behavior
Provide interactive insights using visuals

📂 Dataset Used
The project uses the following datasets:

Students.csv → Student basic details
Scores.csv → Subject-wise marks
Attendance.csv → Daily attendance records
Behavior.csv → Student behavior data

🛠 Tools & Technologies
Microsoft Power BI
DAX (Data Analysis Expressions)
Power Query Editor

🔗 Data Modeling

Relationships created:

Students → Scores (StudentID)
Students → Attendance (StudentID)
Students → Behavior (StudentID)

📊 Key Metrics (DAX Measures)
% Score → Student performance percentage
Average Score → Average marks per subject
Attendance % → Attendance ratio
Behavior Count → Count of behavior types
Total Students → Total number of students

📈 Visualizations Used
📊 Bar Chart → Subject-wise performance
📉 Line Chart → Performance trend over terms
🍩 Donut Chart → Behavior distribution
📋 Table → Detailed student data
🔢 KPI Cards → Summary metrics

🎛 Features Implemented
🔹 Slicers
Class
Section
Subject
Term

🔹 Drillthrough
Navigate to Student Details page
View individual student performance

🔹 Conditional Formatting
Highlight performance using colors
Green → High score
Red → Low score

🔹 Bookmark Navigation
Academic View (Score charts)
Behavior View (Behavior & attendance)
Buttons used for switching views

📱 Additional Features
Interactive dashboard
Clean UI design
User-friendly navigation

📌 Conclusion
This dashboard provides a clear and interactive way to analyze student data. It helps in identifying high performers, low performers, and attendance trends, making it useful for decision-making.
